## Introduction to Binary Translators

An online binary translator is a tool that lets you convert text to binary and binary to text instantly. Binary code is the foundation of computing, consisting of only two digits: 0 and 1. Every character you type is stored in binary form.

**For example:**
- The letter A in binary is `01000001`.
- The letter B in binary is `01000010`.

Using a [binary code decoder online for free](/tools/binary-translator), you can easily see how computers store and process information.

## Code Example – Text to Binary in Python

```python
def text_to_binary(text):
    return ' '.join(format(ord(char), '08b') for char in text)

def binary_to_text(binary):
    return ''.join(chr(int(b, 2)) for b in binary.split())

# Example usage
print(text_to_binary("Hello"))  
# Output: 01001000 01100101 01101100 01101100 01101111

print(binary_to_text("01001000 01100101 01101100 01101100 01101111"))  
# Output: Hello
```
This simple script serves as a binary encoder/decoder.
